# Specification Documents

This directory contains AI-ready specification documents for the QRL Trading Bot project.

## Naming Convention

Files follow the pattern: `spec-[a-z0-9-]+.md`

The name prefix describes the high-level purpose, which is one of:
`schema`, `tool`, `data`, `infrastructure`, `process`, `architecture`, or `design`.

## Usage

Specification files are generated via the `specification` agent or the `/spec` prompt. They define
requirements, constraints, and interfaces in a structured, machine-readable format.
